# 시간 설정

## timezone 변경
설정 가능한 timezone 출력
```console
timedatectl list-timezones
```
```bash
timedatectl list-timezones | grep Seoul
```
```text
Asia/Seoul
```

시간대 변경
```
sudo timedatectl set-timezone Asia/Seoul
```

## NTP 설치 
```
sudo apt install ntp
```

재시작
```
sudo systemctl restart ntp
```

상태 확인
```
sudo systemctl status ntp
```
